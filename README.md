Created in Quatus II 15.0 for the Altera DE1-SoC FPGA board.
The graphics are displayed via the vga_adapter.

###INSTRUCTIONS

    ----- KEYS -----
    KEY(0): Reset game
    KEY(1): Start game
    KEY(2): Drop block

    ----- SWITCHES -----
    Input color and difficulty on switches before pressing KEY[1]
    SW(9:7): Color of blocks
    SW(0):-> Difficulty
